int seabreeze_get_formatted_spectrum_length(int index, int *error_code)

Parameters

int: index

The index of a device previously opened by seabreeze_open_spectrometer

int *: error_code

A pointer to an integer that can be used for storing error codes

Returns

int

An integer denoting the length of a formatted spectrum (In pixels)

Example

// From this point on, "i" will be used as any index of an open device
// This method will essentially help you with other methods like get_wavelengths and get_formatted_spectrum
// It could be called inside the parameters of those methods in place of the "length" or "buffer_length" parameters,
// but it's more resource-conservative to call it once and store it as an int
// If you are using multiple devices, you could store the different lengths in an array with the appropriate indices

int pixels = seabreeze_get_formatted_spectrum_length(i, &error_code);
