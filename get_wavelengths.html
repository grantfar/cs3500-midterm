seabreeze_get_wavelengths(int index, int *error_code, double *wavelengths, int length)

Parameters

int: index

The index of the device you want to get readings from, previously opened with seabreeze_open_spectrometer.

int *: error_code

A pointer to an integer that can be used for storing error codes.

double *: wavelengths

An array of doubles that the wavelengths will be copied to.

int: length

The number of values to copy into the wavelength array.

Returns

int

seabreeze_get_wavelengths returns the number of wavelengths that were written to the wavelengths array.

Example

// Typically, the "length" parameter will be equal to the formatted spectrum length
int pixels = seabreeze_get_formatted_spectrum_length(i, &error_code);

// This is an array of doubles that will be used to store wavelengths
// It should be able to hold as many doubles as the spectrum length
double *wavelengths = malloc(pixels * sizeof(double));

// After using this method, "wavelengths" will be full of the wavelengths over the spectrum length
seabreeze_get_wavelengths(i, &error_code, &wavelengths, pixels);
