Prerequisites

Before we get started, you will need:

  • Linux
    • If you already have another OS installed, you can dual boot Linux or run it on a virtual machine
    • Other OS's can run SeaBreeze, but this tutorial will focus on Linux
    • Info on using these other OS's can be found in the SeaBreeze documentation
  • An Ocean Optics spectrometer
    • Supported spectrometers (As of 2/26/18):
      • STS
      • Spark
      • Flame-S
      • Flame-T
      • Flame-NIR
      • USB2000+
      • USB4000
      • HR2000
      • HR4000
      • Maya2000Pro
      • Maya-LSL
      • QEPro
      • QE65Pro
      • NIRQuest (256/512)
      • Ventana
      • Torus
      • Jaz
  • libusb-dev 0.1
  • gcc or g++

Downloading SeaBreeze

  • Download from the SeaBreeze Product Site
  • Unzip the contents of the download

Compiling Seabreeze

Seabreeze cannot be compiled with newer versions of gcc/g++ without majorly edits of the makefiles/code. To circumvent this we can install older versions. To do so in ubuntu type:
sudo apt install gcc5
This installs gcc5 but we need to set it to default

To set gcc5 as default, we use update-alternatives:
sudo update-alternatives --install /usr/bin/gcc gcc /usr/bin/gcc5 51
Do both the install and alternitives for g++5

Setting Up the Libraries

On the Linux terminal, cd to the "seabreeze-3.0.1.1" directory.
From there, enter:
sudo cp -R SeaBreeze/include/ /usr/local/include/ && sudo mv usr/local/include/include/ /user/local/include/SeaBreeze && sudo cp SeaBreeze/lib/libseabreeze.so /usr/lib && sudo ldconfig -n -v /usr/lib

File Headers

We would recommend finding the file "SeaBreezeWrapper.h" in the SeaBreeze folder and placing it somewhere you can easily access it from the file you are writing. (There is another API "SeaBreezeAPI.h" that is available, this tutorial will be focusing on "SeaBreezeWrapper.h").
If your SeaBreezeWrapper.h file is placed in an "api" directory in the directory your program is in, you would put #include "api/SeaBreezeWrapper.h" in your file headers.
