Prerequisites

Before we get started, you will need:

  • Linux
    • If you already have another OS installed, you can dual boot Linux or run it on a virtual machine
  • An Ocean Optics spectrometer
    • Supported spectrometers (As of 2/26/18):
      • STS
      • Spark
      • Flame-S
      • Flame-T
      • Flame-NIR
      • USB2000+
      • USB4000
      • HR2000
      • HR4000
      • Maya2000Pro
      • Maya-LSL
      • QEPro
      • QE65Pro
      • NIRQuest (256/512)
      • Ventana
      • Torus
      • Jaz
