seabreeze_close_spectrometer(int index, int *error_code)

Parameters

int: index

Just like seabreeze_open_spectrometer, this is the device index for which the method will close the spectrometer from 0 to n-1 for n devices.

int *: error_code

A pointer to an integer that can be used for storing error codes.

Returns

This function always returns 1.

Example

// This example will carry over the openDevices int array from seabreeze_open_spectrometer

for (int i = n - 1; i >= 0; i--) {

if (openDevices[i] == 1) {

seabreeze_close_spectrometer(i, &error_code);

}

}
