Team #15: Grant Farnsworth and James Eckler

We are both members of the Western Aerospace Launch Initiative (WALI) as part of the Control and Data Handling (CDH) subgroup. WALI's goal is to create a nanosatellite that they can send into space, with the specific purpose of using plasma spectroscopy to examine the plasma plume of other satellites for defects. We wrote the code to operate our spectrometer using the libraries and methods provided by SeaBreeze.

What is SeaBreeze?

SeaBreeze is a device driver created by Ocean Optics, an open source version of their Omnidriver software that can be controlled by c/c++. It works on most of their spectrometers.

Why did we create this site?

While there is documentation on SeaBreeze, it can be difficult to comprehend at first glance. We intend to give clear, easy to understand instructions on how to get started with SeaBreeze and write a program that will take readings using an Ocean Optics spectrometer.

What is covered in this tutorial?

In this tutorial, you will be shown how to install SeaBreeze, taught about its essential methods, and given example code on how to use those methods to get readings from your Ocean Optics spectrometer.
