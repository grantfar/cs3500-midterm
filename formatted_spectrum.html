int seabreeze_get_formatted_spectrum(int index, int *error_code, double *buffer, int buffer_length)

Parameters

int: index

The index of a device previously opened with seabreeze_open_spectrometer

int *: error_code

A pointer to an integer that can be used for storing error codes

double *: buffer

A buffer (With memory already allocated) to hold the spectral data

int: buffer_length

The length of the buffer in floats

Returns

int

The number of floats read to the buffer

Example

// To get a reading of the spectrum, you'll need the spectrum length
int pixels = seabreeze_get_formatted_spectrum_length(i, &error_code);

// You will need an array as large as the spectrum length
double *spectrum = malloc(pixels * sizeof(doubles));

// After calling this method, "spectrum" will be full of the spectrum intensity along the wavelengths covered by your spectrometer
// The matching indices of the "wavelengths" array and the "spectrum" array will give you the wavelength and intensity at that wavelength seabreeze_get_formatted_spectrum(i, &error_code, &spectrum, pixels);
